# RPGUser
## 设定
* 宝石：等级，种类，力量，智慧，敏捷  
* 装备：等级，力量、智慧、敏捷  
* 英雄：稀有度，等级，力量，智慧，敏捷  
* 用户：等级，金币，经验  


__宝石__
* 类型，等级
* 总攻击力/总智慧/总敏捷 = 类型默认力量/智慧/敏捷 + 等级*0.5

__装备__
* 等级，力量，智慧，敏捷  
* 总攻击力/总智慧/总敏捷 = 初始力量/智慧/敏捷 + 等级*0.5 + 所有宝石力量/智慧/敏捷

__英雄__
* 等级，HP,力量，智慧，敏捷  
* 总攻击力/总智慧/总敏捷 = 初始力量/智慧/敏捷 + 等级*0.5 + 所有装备力量/智慧/敏捷


__用户__
* 金币，等级
* 总攻击力是所有英雄攻击力的总和
